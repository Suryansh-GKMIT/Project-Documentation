# Authentication and Authorization Flow

This section explains how user authentication and authorization work within the **SkillCheck** platform.<br>
The process ensures that only verified users can access system resources according to their assigned roles (Admin, Test Creator, or Test Taker).<br>

This mechanism ensures secure communication between the frontend and backend, maintaining strict access control across the platform.<br>
